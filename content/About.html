About

請各學員自行準備一副有線耳機, 若使用手機觀看影片, 則可以準備手機適用的有線或無線耳機.

本課程為機械設計工程系大二的電腦輔助設計與實習課程, 課程目標在教導學員如何利用電腦輔助機械設計套件建立零組件模型 (機械元件設計領域, using Solid Edge), 並能轉入機電模擬系統 (using CoppeliaSim) 中配置致動器與感測器後進行控制系統設計 (電機與電子元件設計領域, using 自動控制 and 控制系統設計), 目的是能透過 Web-based 程式系統的整合 (資訊設計領域, using Javascript and Python), 在隨後的協同產品設計實習課程中加以應用.

basic_portable_python.7z (下載 15MB, 解開後 55MB)

下載 Portable_2023_fall_1.3g_5g.7z (需要下載密碼)

extra_dlls.7z (需要下載密碼)

Introdution to Computer Aided Design and Analysis (需要下載密碼)

利用全雲端環境上課:

Ubuntu server、Replit、Github、Onedrive 與 Onshape 上進行電腦輔助設計與實習.

利用全近端環境上課:

可攜 Python 程式系統、可攜 Solid Edge、Solvespace 、NX 與 CoppeliaSim 上進行電腦輔助設計與實習.

參考:

how-to-make-almost-anything-修課經驗分享

https://fab.cba.mit.edu/classes/MAS.863/

課程評分: 線上測驗、期中網站與簡報、期末網站與簡報.

線上測驗: w1 建立帳號後 (請將密碼寫入手機的備忘錄), 每週不定時進行.

期中網站與簡報: 期中考前一週開始進行.

期末網站與簡報: 期末考前一週開始進行.

期中成績: 期中考與之前各週線上測驗成績平均

學期成績: (期中成績與之後各週線上測驗成績平均)*60%+(期中網站與簡報成績)*20%+(期末網站與簡報成績)*20%
