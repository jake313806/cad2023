TCExam

已經配合 TCExam 安裝完成的空白 PostgreSQL 資料庫目錄 data 壓縮檔案為:

data_postgresql_for_tcexam.7z

此檔案位於 pgsql 目錄中, 解開壓縮檔案可以得到 pgsql\data 目錄, 只要再利用 pgsql\start.bat 就可以啟動資料庫伺服器.

其次 nginx 與 php 服務啟動後就可以正常登入 TCExam.

nginx http 伺服器的設定:

    server {
       # orig
       # listen       [::]:80 default ipv6only=on;
	listen 80;
	listen [::]:80;
        server_name  exam.cycu.org;
        #return       301 https://$server_name$request_uri;
        
        allow 2001:288:6004:17::0/32;
        #deny all;
	allow 140.130.17.0/24;
	deny all;
	
	return 302 https://exam.cycu.org;
        
        #location /{
            #root C:/tcexam/downloads;
            #auth_basic "For Authorized Users Only!";
            #auth_basic_user_file C:/tcexam/nginx-1.20.1/.htpasswd;
        #}
    }

其中只允許特定 IP 範圍的電腦連線, 且透過 return 302 直接轉向 https

而 nginx https 伺服器設定:

server { 
    # orig
    #listen    [::]:443 default ssl ipv6only=on;
    listen 443 default ssl;
    listen [::]:443 default ssl;
    
    root C:/tcexam/tcexam;

    server_name    exam.cycu.org;
    ssl_certificate    C:/tcexam/fullchain.pem;
    ssl_certificate_key    C:/tcexam/privkey.pem;
    ssl_protocols    TLSv1 TLSv1.1 TLSv1.2;
    ssl_ciphers    HIGH:!aNULL:!MD5;
    
    allow 2001:288:6004:17::0/32;
    #deny all;
    allow 140.130.17.0/24;
    deny all;
    
    location /{ 
        index index.html index.php;
    }
    
    location ~ \.php$ {
        fastcgi_pass   127.0.0.1:9123;
        fastcgi_index  index.php;
        fastcgi_param  SCRIPT_FILENAME  $document_root$fastcgi_script_name;
        include        fastcgi_params;
    }
}

其中除了使用制式數位簽章外, 同時透過 fastcgi 啟動 執行 php 程式, 而 fastcgi 伺服器則以 php 作為服務名稱啟動.

名稱命名為 php 的服務指令: C:\tcexam\php8.1.7\php-cgi.exe -b 127.0.0.1:9123

表示啟動後, 將以近端 localhost 埠號 9123 啟動.
